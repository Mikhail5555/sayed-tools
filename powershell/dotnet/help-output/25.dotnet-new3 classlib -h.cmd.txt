Usage: dotnet new classlib [templateoptions] [options]

Class library
Author: Microsoft
Options:
  -h, --help          Displays help for this command.
  --force             Forces content to be generated even if it would change existing files.
  -v, --verbosity     Set the verbosity level of the command. Allowed values are q[uiet], m[inimal], n[ormal], d[etailed], and diag[nostic].

Template Options:
  -n, --name          The name for the output being created. If no name is specified, the name of the current directory is used.
  -o, --output        Location to place the generated output.
  -lang, --language   Specifies the language of the template to create.
                          C#
                          F#
                          VB
                      Default: C#
  -f|--framework  The target framework for the project.                              
                      netcoreapp2.0, netcoreapp1.1, netcoreapp1.0, netstandard2.0, netstandard1.6, netstandard1.5
                      netstandard1.4, netstandard1.3, netstandard1.2, netstandard1.1, netstandard1.0                         
                  Default: netstandard2.0
  --no-restore    If specified, skips the automatic package restore of the project on create.
                  Type: bool (Optional)                                                    
                  Default: false
